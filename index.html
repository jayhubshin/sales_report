📋 Supabase 무료 설정 방법 (5분)
1. supabase.com → 회원가입 → New project 생성
2. 좌측 Storage → New bucket → 이름: epc-data
   → Public bucket 체크 ON (파일 읽기 허용)
3. 좌측 Settings → API
   → Project URL 복사 → 아래 URL에 붙여넣기
   → anon public 키 복사 → 아래 Anon Key에 붙여넣기
4. 연결 테스트 → 저장
anon/public 키를 사용합니다. service_role 키는 사용하지 마세요.
