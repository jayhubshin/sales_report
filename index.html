Classic Token 발급 방법 (권장)
1. GitHub.com → 우측 상단 프로필 → Settings
2. 좌측 하단 Developer settings 클릭
3. Personal access tokens → Tokens (classic)
4. Generate new token (classic) 클릭
5. repo 체크박스 전체 선택 → Generate
6. 생성된 ghp_... 토큰을 복사하여 아래에 붙여넣기
Fine-grained Token 발급 방법
1. GitHub.com → Settings → Developer settings
2. Personal access tokens → Fine-grained tokens
3. Generate new token 클릭
4. Repository access: 해당 저장소 선택
5. Permissions → Contents: Read and write 선택
6. 생성된 github_pat_... 토큰 복사
저장소 내 엑셀이 저장될 폴더 (없으면 자동 생성)
